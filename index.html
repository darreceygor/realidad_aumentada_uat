Lenguajes

Como se vio antes las páginas web son de desde un punto de vista solo información en texto que está marcado y presenta reglas que indican el estilo que debe de asignársele a la información y elementos presentes en el documento que los contienen, estas etiquetas y reglas de estilo se realizan por medio de lenguajes de marcado de texto y lenguajes para especificar el estilo que se deberá aplicar en los diversos tipos de archivos web, los lenguajes usados para realizar lo antes mencionado son HTML y CSS.


HTML son las siglas de HyperText Markup Language, es decir, lenguaje de marcado de hipertexto, este constituye un estándar asumido y reconocido a nivel mundial. Sus normas las establece una entidad sin fines de lucro llamada World Wide Web Consortium (W3C) que lo define como «un lenguaje reconocido universalmente y que permite publicar información de forma global».


En cuanto a CSS (del inglés Cascading Style Sheets u “hojas de estilo en cascada”), es el lenguaje de que determina el aspecto con el que se visualiza el contenido de una página web. Con este se puede especificar aspectos como el color, estilos de letra o modificar características de las imágenes, entre muchos otros aspectos más, este puede estar tecleado en el mismo archivo en el que se teclean las etiquetas HTML, pero es costumbre el que este se encuentre separado en otro archivo diferente que contenga todas las demas reglas que dictan el aspecto de una página web un archivo con reglas de estilo CSS y un archivo con la información marcada con etiquetas HTML es lo que se combina para crear las páginas que se visualizan en los diversos exploradores de internet.

HTML

El lenguaje de marcado de hipertexto se basa en “distintivos” conocidos como etiquetas o tags con los cuales estructura el texto presente en las páginas web y los diversos elementos presentes en estas. Estos se en la mayoría de los casos se colocan antes y después de cada elemento al que definen (en este caso con una etiqueta de apertura y una de cierre), su característica principal es que se indican entre corchetes/menor que mayor que (<....>).


Algunos ejemplos de etiquetas HTML son:


html: indica el inicio del documento HTML.


head: contiene información de la página que no es visible para los visitantes, pero sirve para que los navegadores identifiquen lo que hay en ella.


body: es la parte visible de la página web que se visualizara en la ventana de los navegadores/ exploradores web, en esta se coloca la información y elementos visualizables incluidos textos, imágenes, tablas, etc.


div: divide el contenido en bloques separados por espacios. Estos bloques pueden tener párrafos, imágenes, vídeos, etc. A veces una misma etiqueta div tiene otras dentro de ella.


br: este es un ejemplo de etiqueta que como tal no tiene una etiqueta de apertura y otra de cierre, siendo una sola etiqueta sin contenido, esta etiqueta crea un salto de línea, pero sin dejar un espacio de por medio.


h1, h2…h6 : marcan los títulos y subtítulos dentro del documento y sirven para jerarquizar la información, así como indicar a los buscadores lo que contiene cada apartado.


a: define los hipervínculos o links dentro y fuera de la web.


p: el texto entre las etiquetas de apertura y cierre de tipo “p” se representará como agrupado en un párrafo.


i: el texto entre las etiquetas de apertura y cierre de tipo “i” se representará como letra cursiva.


b y strong: el texto entre las etiquetas de apertura y cierre de tipo “b” o “strong” se representará como letra en negrita.


u: el texto entre las etiquetas de apertura y cierre de tipo “u” se representará como texto subrayado.


Los anteriores son solo un ejemplo de las muchas etiquetas HTML existentes.

CSS

El mecanismo CSS se basa en “reglas de estilo” (o simplemente “reglas”) que permiten dar formato a los distintos elementos del lenguaje HTML. Estas reglas se estructuran de una manera específica con una serie de componentes los cuales son:.


Selector: identifica en que parte de la información marcada con HTML es donde se aplicaran los cambios, usándose para este fin la etiqueta especifica presente en el archivo HTML, o identificadores especiales con los cuales poder afectar a varia información presente dentro del archivo HTML, estos identificadores son los “id’s (id)” y las “classes (class)”. Después de indicar el selector, las reglas/atributos/propiedades que se desea establecer van dentro de corchetes.


Propiedad: se refiere al atributo que se va a modificar, ejemplos podrían ser ser el color o el tipo de fuente, entre muchos otros más. Para aplicar estas propiedades a elementos en el archivo HTML se puede emplear por ejemplo la etiqueta “p” como selector y de esta manera poder afectar a todos los párrafos en un archivo HTML, o referirse solamente a un párrafo o parte específica del archivo HTML por medio de un “id”, o a varios párrafos o partes especificas a las que se desea asignar características comunes de diseño empleando una “class” en las partes del archivo HTML deseadas.


Valor: es el valor que se le asignara a la característica de la propiedad. Siguiendo con el ejemplo anterior, podríamos referirnos en el caso de la propiedad color al verde, azul, amarillo, rojo, etc., o en el caso del tipo de fuente a Arial, Helvética, etc., después de establecer el valor se debe de terminar con “;” para especificar que se ha terminado de asignar valor a la regla en cuestión.
